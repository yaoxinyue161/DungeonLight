# Dungeon Light

A dungeon roguelike game built with Pygame.

## Features
- BSP dungeon generation
- A* enemy pathfinding  
- Dynamic torch lighting with shadowcasting
- Fog of war system
- Crystal collection mechanics
- Multiple difficulty modes
- Health and combat system

## Controls
- **WASD / Arrow Keys** - Move
- **Left Mouse Button** - Attack
- **ESC** - Pause / Quit
- **P** - Pause
- **R** - Restart (when game over)

## Installation

```bash
pip install -r requirements.txt
python main.py
